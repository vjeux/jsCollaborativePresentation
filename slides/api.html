}

Method Chaining

function Object() {}
Object.prototype = {
  method: function () {
    // Do something
    return this;
  }
};

var obj = new Object();
obj.method().method();

Keymaster JS

key('shift + a', 'ctrl + alt + 1', function (event, handler) {
	console.log(handler.shortcut); // 'shift + a'
	console.log(handler.keys); // {shift: true, a: true}
});
  • Event Driven Programming: Lambda functions makes it easy to write.
  • Strings are good for introspection
  • Variadic arguments, function as the last argument
